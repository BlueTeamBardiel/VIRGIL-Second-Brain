## Session — 2025-11-12 (evening)

### Security+ Study — CWE-77 Command Injection (1h 20m)

Feynman session via `/cysa`. Hit gaps on:
- Distinction between CWE-77 (OS command injection) vs CWE-89 (SQL injection) — both inject into an interpreter, different targets
- Shell metacharacter taxonomy: which chars are dangerous in which shells
- `subprocess.run(..., shell=True)` vs `subprocess.run([...])` — the difference matters

Key takeaways:
- Always use argument arrays over shell strings when calling subprocesses
- `shlex.quote()` is your last resort if you genuinely must pass user input to a shell
- Input allowlisting beats escaping — if a field should contain a port number, validate `^[0-9]{1,5}$` and reject anything else at the boundary
