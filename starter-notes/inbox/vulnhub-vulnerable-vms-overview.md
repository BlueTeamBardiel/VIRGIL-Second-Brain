# VulnHub Vulnerable VMs Overview

VulnHub is a platform providing free vulnerable virtual machines for penetration testing practice and security research. The site hosts a curated collection of intentionally vulnerable boxes spanning various difficulty levels and security concepts.

## Platform Overview

**Purpose**: Community-driven repository of vulnerable VMs designed for hands-on security training and CTF-style challenges.

**Key Features**:
- Free downloadable vulnerable machines
- Difficulty ratings: Easy, Medium, Hard
- Web machines and standalone boxes
- Community submissions from security researchers
- Author profiles and machine details

## Notable Machine Series

### Matrix-Breakout Series
- **Morpheus:1** — Themed throwback to first Matrix film; players assume Trinity role investigating locked computer
- Related to broader Matrix-themed challenge set

### The Planets Series
- **Earth** — Easy difficulty, two flags (user + root), includes MD5 hashes
- **Mercury** — Baseline easy difficulty reference

## Additional Machines

- **Empire: Breakout** / **Empire: LupinOne** — Medium difficulty, CTF-style enumeration-focused
- **ICA: 1** — Espionage-themed, multi-layer security challenge
- **digitalworld.local: electrical** — VAPT deployment scenario, multiple privilege escalation paths
- **Jangow: 1.0.1** — Emphasis on enumeration
- **Red: 1**, **Napping: 1.0.1**, **Thales: 1**, **Noob: 1** — Various difficulty levels
