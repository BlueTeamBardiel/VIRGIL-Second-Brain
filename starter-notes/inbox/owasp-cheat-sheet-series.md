# OWASP Cheat Sheet Series

A comprehensive collection of concise, high-value security guidance on specific application security topics created by OWASP professionals. Provides practical cheat sheets covering authentication, cryptography, injection prevention, and more.

## Overview

The OWASP Cheat Sheet Series is a community-driven project offering expert-authored security references in easy-to-read format. Resources cover both traditional web security and modern topics like cloud, microservices, and AI.

## Key Topics Covered

- **Authentication & Authorization**: [[Authentication]], [[Authorization]], [[Multi-Factor Authentication]], [[OAuth2]], [[SAML Security]]
- **Web Application Security**: [[Cross-Site Scripting Prevention]], [[Cross-Site Request Forgery Prevention]], [[SQL Injection Prevention]], [[Injection Prevention]]
- **Cryptography & Data Protection**: [[Cryptographic Storage]], [[Password Storage]], [[Key Management]], [[Transport Layer Security]]
- **Infrastructure & Cloud**: [[Docker Security]], [[Kubernetes Security]], [[Secure Cloud Architecture]], [[Infrastructure as Code Security]]
- **Modern Technologies**: [[GraphQL]], [[REST Security]], [[gRPC Security]], [[WebSocket Security]], [[Serverless FaaS Security]]
- **Advanced Topics**: [[LLM Prompt Injection Prevention]], [[AI Agent Security]], [[Software Supply Chain Security]], [[Zero Trust Architecture]]
