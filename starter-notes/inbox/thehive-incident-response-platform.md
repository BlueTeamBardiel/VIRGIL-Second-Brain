# TheHive Incident Response Platform

TheHive is a collaborative security case management platform developed by [[StrangeBee]] and trusted by incident responders, SOCs, CERTs, and CSIRTs worldwide for unified incident response and threat management.

## Overview

TheHive serves as a centralized case management platform that integrates with existing security tools and provides automated analysis and response capabilities through its [[Cortex]] automation engine.

## Core Products

- **TheHive (On-Premises)**: Self-hosted incident response platform
- **TheHive Cloud Platform**: SaaS version hosted on dedicated AWS infrastructure
- **Cortex**: Automated analysis and response engine integrated with TheHive
- **Cloud Images**: IaaS deployment option for cloud-native environments

## Key Features

### Alert Management
- Automatic alert ingestion from security platforms
- Triaging, filtering, and automated response workflows
- Multi-channel notifications ([[Slack]], [[Mattermost]], email, [[Webhooks]], custom HTTP)

### Case Enrichment
- Customizable reports and evidence attachment
- Custom fields and metrics
- Malware sample storage with password-protected ZIP archives
- Tag-based [[threat intelligence]] feeds
- IOC flagging and sighting tracking

### Collaboration
- Multi-organization and team support
- Configurable role-based access control
- Dedicated or collaborative investigation modes
- Real-time incident dashboard
- Task assignment and live action visibility
