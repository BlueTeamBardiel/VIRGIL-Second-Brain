# CompTIA Security+

CompTIA Security+ is a vendor-neutral certification validating IT security skills across threat management, vulnerability assessment, cryptography, and identity/access control. It serves as a foundation-level credential for security professionals and is often required for government and defense contractor roles.

## Overview

**Certification Level:** Entry to mid-level security professional

**Key Focus Areas:**
- [[Threats and Vulnerabilities]] management
- [[Cryptography]] fundamentals
- [[Identity and Access Management]]
- [[Network Security]] concepts
- [[Incident Response]] and recovery
- [[Risk Management]] principles
- [[Security Operations]] and monitoring

## Exam Details

- **Exam Code:** SY0-701 (latest version)
- **Format:** Multiple choice and performance-based questions
- **Duration:** 90 minutes
- **Passing Score:** ~750/900
- **Prerequisites:** CompTIA A+ and Network+ recommended; DoD 8570 requirements often mandate this cert

## Target Audience

- IT professionals transitioning to security roles
- System and network administrators
- Government/military contractors (DoD 8570 compliance)
- Help desk staff advancing to security positions
