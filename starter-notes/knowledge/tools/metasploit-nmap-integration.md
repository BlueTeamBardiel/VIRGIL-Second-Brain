# Metasploit & Nmap Integration

Guide to integrating [[Nmap]] scan results with [[Metasploit]] for streamlined vulnerability assessment and exploitation workflows.

## Overview

Metasploit documentation covers how to import [[Nmap]] scan data into Metasploit to automatically populate targets and leverage discovered services for exploitation.

## Key Integration Points

- **Importing Nmap Results**: Load scan output directly into Metasploit's database
- **Target Population**: Automatically identify hosts and services from scans
- **Exploit Selection**: Match discovered services to appropriate [[Metasploit]] modules
- **Workflow Optimization**: Reduce manual enumeration steps
