# Burp Suite Certified

## What it is
Like a master locksmith who has demonstrated they can pick, probe, and map every tumbler in a complex lock system, a Burp Suite Certified practitioner has proven hands-on proficiency with PortSwigger's web application security testing platform. The **Burp Suite Certified Practitioner (BSCP)** is an exam-based certification by PortSwigger that validates a candidate's ability to identify and exploit real-world web vulnerabilities using Burp Suite Professional as their primary tool.

## Why it matters
During a web application penetration test against an e-commerce platform, a BSCP-level tester would use Burp's Intruder to brute-force hidden parameters, Scanner to identify an XXE injection point, and Repeater to manually craft payloads that bypass a WAF — chaining these findings into a full account takeover. This certification signals to employers that the candidate can conduct structured, methodical web application assessments rather than just running automated scans.

## Key facts
- The BSCP exam is a **4-hour practical exam** requiring candidates to compromise two applications from scratch with no multiple choice — pure exploitation
- Candidates must demonstrate proficiency across **all major web vulnerability classes** covered in PortSwigger Web Security Academy (SQLi, XSS, SSRF, IDOR, XXE, deserialization, etc.)
- Burp Suite **Proxy** is the core tool — it intercepts HTTP/HTTPS traffic between browser and server, enabling manual inspection and manipulation
- **Burp Collaborator** is tested heavily; it detects out-of-band interactions for blind SSRF and blind XXE vulnerabilities
- PortSwigger's **Web Security Academy** (free) is the official preparation path and maps directly to exam content
