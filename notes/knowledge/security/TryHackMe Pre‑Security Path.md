# TryHackMe Pre‑Security Path

An introductory learning path on TryHackMe designed to prepare learners for cybersecurity fundamentals before advancing to more specialized topics. This path covers foundational concepts essential for aspiring security professionals.

## Overview

The Pre‑Security path is TryHackMe's entry‑level curriculum that establishes core knowledge in networking, systems administration, and security principles. It serves as a prerequisite for specialized tracks like CCNA, ethical hacking, and advanced penetration testing.

## Learning Objectives

- Understand fundamental networking concepts and protocols
- Learn basic system administration and Linux/Windows fundamentals
- Explore introductory cybersecurity principles
- Build foundational knowledge for advanced certifications

## What Is It? (Feynman Version)

Imagine a toolbox that teaches you how to solder wires, install a door lock, and check the house’s smoke detector before you learn how to lock the vault against burglars. The Pre‑Security path is that toolbox; it gives you the basics of networking, operating systems, and security so you can move on to deeper topics.

## Why Does It Exist?

Before the path existed, many students jumped straight into specialized labs without knowing why a firewall exists or how an IP address works. That gap led to wasted hours on misconfigured machines, missed exam objectives, and teams deploying insecure systems. The path fixes that by ensuring everyone shares the same baseline, just like a school that requires a basic math class before tackling calculus.

## How It Works (Under the Hood)

1. **Structure**: The curriculum is divided into themed modules—Networking, Linux, Windows, Security Fundamentals.  
2. **Labs**: Each module contains hands‑on labs that run in isolated virtual machines.  
3. **Quizzes**: After every lab you take a short quiz that checks conceptual understanding.  
4. **Progression**: You unlock the next module only after scoring at least 80 % on the current one, forcing mastery.  
5. **Community**: A built‑in discussion forum lets learners compare notes and share solutions, reinforcing learning through teaching.  
6. **Analytics**: The platform tracks completion time and quiz scores, giving you a personal report to adjust study habits.

## What Breaks When It Goes Wrong?

- **Learners**: Without the baseline, they hit confusing error messages and feel lost, leading to frustration and drop‑out.  
- **Employers**: Graduates enter the field without solid fundamentals, causing them to misinterpret logs or misconfigure firewalls, which in turn can open vulnerabilities.  
- **Certifiers**: Exams that assume basic knowledge (e.g., Security+ objective 1.2 “Networking”) become harder to pass, inflating failure rates.  
- **Industry**: Incomplete knowledge means teams spend more time troubleshooting issues that could have been prevented, costing money and exposing data.
