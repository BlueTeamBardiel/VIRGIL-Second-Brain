# CySA+ Cheat Sheet

## What it is
Think of it as a surgeon's pre-op checklist — every critical procedure, tool, and term condensed onto one laminated card before you scrub in. A CySA+ Cheat Sheet is a condensed reference document covering the core domains of CompTIA's Cybersecurity Analyst certification: threat intelligence, vulnerability management, incident response, and security architecture.

## Why it matters
During a live SOC triage, an analyst receives an alert for lateral movement across subnets. Without internalizing frameworks like the Diamond Model or the MITRE ATT&CK matrix — both CySA+ staples — the analyst may chase the wrong indicators and miss the actual persistence mechanism. A solid cheat sheet means these frameworks are second nature, not a memory scramble under pressure.

## Key facts
- **Five CySA+ Domains**: Threat and Vulnerability Management (22%), Software and Systems Security (18%), Security Operations and Monitoring (25%), Incident Response (22%), Compliance and Assessment (13%)
- **Key Tools to Know**: Nessus/Qualys (vulnerability scanning), Wireshark (packet analysis), Splunk/SIEM (log correlation), Nmap (network reconnaissance)
- **Incident Response Phases (PICERL)**: Preparation → Identification → Containment → Eradication → Recovery → Lessons Learned
- **Threat Intelligence Taxonomy**: Strategic (executive decisions), Tactical (TTPs for defenders), Operational (active campaigns), Technical (IOCs like hashes/IPs)
- **CVSS Scoring**: Base score 0–10; Critical = 9.0–10.0; factors include Attack Vector, Complexity, Privileges Required, and User Interaction
