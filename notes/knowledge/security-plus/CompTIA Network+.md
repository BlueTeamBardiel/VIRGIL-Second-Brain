# CompTIA Network+

## What it is
Like a driver's license before you attempt Formula 1 racing, Network+ certifies you've mastered the road rules of networking before tackling advanced security work. It's a vendor-neutral CompTIA certification validating foundational knowledge of network infrastructure, protocols, troubleshooting, and security concepts. It sits between CompTIA A+ and Security+ in the certification pathway.

## Why it matters
During a ransomware incident response, an analyst who lacks Network+ fundamentals may fail to interpret a Wireshark capture showing lateral movement over SMB port 445 — missing the attacker's pivot from a compromised workstation to a domain controller. Understanding subnetting, VLANs, and routing protocols isn't academic; it's the difference between containing a breach in minutes versus hours. Network segmentation strategies — a core Network+ topic — directly limit blast radius when an endpoint is compromised.

## Key facts
- **Exam code N10-009** covers five domains: Networking Concepts (23%), Network Implementation (19%), Network Operations (17%), Network Security (20%), and Network Troubleshooting (21%)
- Validates knowledge of OSI and TCP/IP models, IPv4/IPv6 subnetting, DNS, DHCP, and common ports (e.g., SSH=22, HTTPS=443, RDP=3389)
- Covers physical and logical network topologies, including SD-WAN, VLAN configuration, and wireless standards (802.11ax/Wi-Fi 6)
- Network security topics include firewall types, IDS/IPS placement, NAC, VPN protocols (IPSec, SSL/TLS), and the CIA triad applied to network design
- Certification is valid for **3 years**; renewal via Continuing Education Units (CEUs) or passing a new exam; recognized as a DoD 8570 baseline certification for IAT Level I
