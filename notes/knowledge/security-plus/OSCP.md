# OSCP

## What it is
Think of it like a 24-hour cooking competition where you're locked in a kitchen and must actually produce dishes — not just recite recipes. The Offensive Security Certified Professional (OSCP) is a hands-on penetration testing certification from Offensive Security that requires candidates to compromise multiple machines in a live lab environment within 24 hours, then write a professional penetration test report. It is widely regarded as the gold standard for proving real-world offensive security skills, not just theoretical knowledge.

## Why it matters
When a company hires a penetration tester to validate their network defenses, they need confidence that the tester can actually exploit vulnerabilities — not just run automated scanners. OSCP holders have demonstrated they can manually identify misconfigurations, chain exploits together, escalate privileges, and pivot through networks, making the certification a trusted hiring signal for red team and pentesting roles. Organizations like CISA specifically list hands-on certifications like OSCP as preferred credentials for offensive security positions in federal contractor frameworks.

## Key facts
- Requires exploiting a minimum number of machines (typically 3 of 6) in a 24-hour exam lab, plus a standalone Active Directory set
- The core methodology enforced is "Try Harder" — manual enumeration and exploitation over automated tool dependency
- The accompanying course is **PEN-200** (formerly PWK — Penetration Testing with Kali Linux), which covers buffer overflows, web attacks, Active Directory, and privilege escalation
- Exam machines are scored by points (10, 20, or 25 pts each); a passing score is 70 out of 100
- A professional pentest report must be submitted within 24 hours after the exam concludes — the technical skill alone is not enough
