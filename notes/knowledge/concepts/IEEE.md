# IEEE

## What it is
Think of IEEE like the United Nations for engineers — a global body where technical nations (industries, academics, researchers) negotiate and publish the rules everyone agrees to follow. Formally, the Institute of Electrical and Electronics Engineers is a professional organization that develops and publishes widely adopted technical standards, including many foundational to networking and cybersecurity.

## Why it matters
When an attacker exploits a vulnerability in WPA2 Wi-Fi (like the KRACK attack in 2017), they're targeting flaws in **IEEE 802.11i** — the specific standard defining Wi-Fi security. Without IEEE standardizing 802.11, every vendor would implement wireless differently, making both interoperability and consistent security patching nearly impossible. IEEE standards create a single, auditable target for both attackers and defenders.

## Key facts
- **IEEE 802.11** defines the Wi-Fi family of standards (802.11a/b/g/n/ac/ax), including security protocols WEP, WPA, WPA2, and WPA3
- **IEEE 802.1X** defines port-based Network Access Control (NAC), the authentication framework used in enterprise Wi-Fi and wired networks with RADIUS
- **IEEE 802.3** defines Ethernet — the physical and data link layer standard for wired LANs
- **IEEE 802.1Q** defines VLAN tagging; misconfiguration enables VLAN hopping attacks
- IEEE is distinct from IETF (which manages RFCs for internet protocols like TLS and IPsec) — IEEE handles hardware/physical/data-link layer standards
