# SY0-701

## What it is
Like a driver's license renewal that adds new road rules (roundabouts, electric vehicle lanes) to reflect how driving has changed, SY0-701 is the updated version of CompTIA's Security+ certification exam, replacing SY0-601 in November 2023. It validates foundational cybersecurity skills with an emphasis on modern threats, hybrid environments, and automation-driven security operations.

## Why it matters
A security analyst studying for SY0-701 will encounter scenario-based questions about responding to a ransomware incident in a cloud-hybrid environment — a gap that SY0-601 underrepresented. The updated exam directly maps to real job tasks like using SIEM dashboards, implementing zero trust architecture, and hardening containerized workloads, making certified professionals more immediately useful on day one of employment.

## Key facts
- **Five domains:** General Security Concepts (12%), Threats/Vulnerabilities/Mitigations (22%), Security Architecture (18%), Security Operations (28%), Security Program Management & Oversight (20%)
- **Format:** Maximum 90 questions, mix of multiple choice and performance-based questions (PBQs); 90-minute time limit
- **Passing score:** 750 on a scale of 100–900
- **New emphasis areas vs. SY0-601:** Zero trust, vendor/supply chain risk, AI/ML threat vectors, and security automation (SOAR) are significantly expanded
- **Validity:** Certification is valid for 3 years; renewal via CE credits or retesting; counts toward DoD 8570/8140 IAT Level II requirements
