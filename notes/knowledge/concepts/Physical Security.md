# Physical Security

## What it is
Think of physical security as the moat, drawbridge, and guards around a castle — all the digital firewalls in the world mean nothing if an attacker can walk up to your server and pull the hard drive out. Physical security encompasses the controls, barriers, and procedures that restrict unauthorized physical access to systems, facilities, and sensitive assets.

## Why it matters
In 2013, Edward Snowden demonstrated that even the NSA's formidable cyber defenses couldn't stop an insider who simply walked out with classified data on USB drives. A robust physical security posture — including strict badge access to server rooms, USB port blockers, and CCTV surveillance with motion logging — would have created multiple friction points that could have detected or prevented the exfiltration.

## Key facts
- **Mantrap (airlock)** — a double-door entry system where the first door must close before the second opens, preventing tailgating; a direct exam favorite for controlling physical access
- **Faraday cages** block electromagnetic signals, preventing RF-based attacks (like key fob relay attacks) and unauthorized wireless communication from secure areas
- **Cable locks, server cage locks, and tamper-evident seals** are compensating controls when full room access control isn't possible
- **Clean desk policy** is classified as an administrative physical control — it prevents shoulder surfing and document theft
- **Defense-in-depth layers** for physical security: perimeter (fences, bollards) → building (locks, guards) → room (badge readers, biometrics) → device (cable locks, encryption)
