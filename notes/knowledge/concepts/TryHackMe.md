# TryHackMe

## What it is
Think of it like a flight simulator for cybersecurity — you learn to fly dangerous aircraft without risking a real crash. TryHackMe is a browser-based, gamified cybersecurity training platform that provides guided "rooms" (self-contained learning modules) covering offensive and defensive security topics, each with pre-configured virtual machines accessible directly in the browser via OpenVPN or AttackBox.

## Why it matters
A SOC analyst candidate who has never touched a real SIEM or run a port scan is a liability on day one. TryHackMe bridges that gap by letting learners practice real tools — Nmap, Metasploit, Burp Suite, Splunk — against intentionally vulnerable machines in a legal, isolated environment, building the muscle memory that certifications like Security+ test conceptually but rarely reinforce hands-on.

## Key facts
- Rooms are categorized by difficulty (Easy/Medium/Hard) and domain (e.g., Network Security, Web Exploitation, Forensics, SOC Level 1), making structured learning paths possible for cert prep.
- The platform uses a point-and-streak gamification system; completing rooms earns XP and ranks, which research shows improves skill retention versus passive study.
- Learning paths like **SOC Level 1** and **Jr Penetration Tester** map directly to entry-level job roles and align with CompTIA CySA+/PenTest+ objectives.
- AttackBox provides an in-browser Kali Linux instance, removing the barrier of local VM setup — critical for learners on locked-down corporate machines.
- Free-tier access exists, but most guided learning paths and advanced rooms require a paid subscription (~$14/month).
