# CCNP Security

## What it is
Like a master locksmith who doesn't just install deadbolts but designs entire building security systems — blueprints, alarm integration, and access control — CCNP Security certifies engineers who architect and operate comprehensive network security infrastructures. It is Cisco's professional-level certification validating advanced skills in securing networks using Cisco technologies, sitting one tier below the expert-level CCIE Security.

## Why it matters
When a Fortune 500 company suffers a lateral movement attack — where an adversary pivots from a compromised endpoint through the network toward a database server — a CCNP Security-certified engineer is equipped to implement Cisco's Zero Trust segmentation using ISE (Identity Services Engine), deploy Firepower NGFW policies to block east-west traffic, and configure StealthWatch to detect anomalous internal flows before data exfiltration occurs.

## Key facts
- **Core exam required**: SCOR 350-701 (Implementing and Operating Cisco Security Core Technologies) covers network security, cloud security, content security, endpoint protection, and secure network access
- **Concentration exams**: Candidates choose one specialty — SASE, Firewall, Identity Management, Email Security, or Web Content Security — to complete the certification
- **No prerequisite certification** is formally required, though CCNA Security knowledge is assumed
- **Recertification**: Required every three years via continuing education credits or retaking the exam
- **Maps to real tools**: Covers Cisco-specific platforms including ASA/Firepower, Umbrella, Duo (MFA), ISE, and AMP for Endpoints — not just abstract concepts
