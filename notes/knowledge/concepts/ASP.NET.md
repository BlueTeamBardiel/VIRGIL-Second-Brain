# ASP.NET

## What it is  
Think of ASP.NET as a modular kitchen where each appliance can be plugged in to cook a dish. It’s a server‑side web application framework from Microsoft that lets developers build dynamic pages and APIs using .NET languages like C# and VB.NET.
