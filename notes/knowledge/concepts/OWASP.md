# OWASP

## What it is
Think of OWASP as the Consumer Reports of web application security — an independent nonprofit that tests, documents, and publishes unbiased findings so developers and defenders can make informed decisions. The Open Worldwide Application Security Project produces freely available research, tools, and standards focused on improving software security across the industry.

## Why it matters
A development team ships a new login portal without input sanitization. An attacker exploits a SQL injection flaw — ranked #3 in the OWASP Top 10 — to dump the entire user credential database in under two minutes. Had the team followed OWASP's secure coding guidelines and used their testing checklist (WSTG), the vulnerability would have been caught before production deployment.

## Key facts
- The **OWASP Top 10** is the industry-standard awareness document for critical web application security risks, updated periodically (most recent: 2021); Broken Access Control currently holds the #1 spot
- OWASP publishes the **Web Security Testing Guide (WSTG)**, a comprehensive manual for penetration testing web applications
- **OWASP ZAP (Zed Attack Proxy)** is a free, open-source web application scanner used for both manual and automated vulnerability testing
- The **OWASP ASVS (Application Security Verification Standard)** provides a framework of security requirements for designing, developing, and testing web applications — commonly referenced in compliance contexts
- OWASP is **vendor-neutral and free** — all documentation and many tools are available without cost, making it a baseline reference for Security+ and CySA+ exam content
