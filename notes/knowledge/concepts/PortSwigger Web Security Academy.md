# PortSwigger Web Security Academy

## What it is
Think of it as a flight simulator for web hackers — a free, browser-based lab environment where you practice real attacks against deliberately vulnerable applications without touching anything illegal. Precisely, it is an online training platform created by the makers of Burp Suite that delivers structured learning paths covering OWASP-aligned vulnerabilities through guided labs, explanations, and community solutions.

## Why it matters
When security teams perform web application penetration tests, they need practitioners who can identify vulnerabilities like SQL injection, SSRF, and insecure deserialization under time pressure. PortSwigger Academy trains analysts to recognize these attack patterns hands-on, meaning a CySA+ candidate who has completed its labs arrives at an engagement already knowing how attackers chain IDOR with privilege escalation to exfiltrate data — not just that such chains theoretically exist.

## Key facts
- Covers all major OWASP Top 10 categories with dedicated labs, including SQL injection, XSS, CSRF, XXE, SSRF, access control flaws, and business logic vulnerabilities
- Labs run entirely in-browser against isolated target applications, requiring no local VM setup — Burp Suite Community Edition integrates directly
- Offers three difficulty tiers per topic: Apprentice, Practitioner, and Expert, with Expert labs reflecting real-world CTF-level complexity
- The platform is **completely free**, making it the de facto standard for web application security self-study referenced in CySA+/OSCP preparation guides
- PortSwigger's research team publishes original vulnerability disclosures (e.g., HTTP request smuggling, web cache poisoning), and Academy labs often teach those exact novel techniques
