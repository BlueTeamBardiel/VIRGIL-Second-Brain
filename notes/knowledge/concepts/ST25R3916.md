# ST25R3916

## What it is
Think of it as a Swiss Army knife jammed into the pocket of every NFC-enabled device — capable of reading, writing, and even eavesdropping on contactless conversations. The ST25R3916 is a high-performance NFC/RFID front-end IC manufactured by STMicroelectronics, supporting all four NFC Forum device types (reader/writer, P2P, card emulation) across ISO 14443, ISO 15693, and ISO 18092 protocols. It operates at 13.56 MHz and is commonly embedded in smartphones, payment terminals, and access control readers.

## Why it matters
Security researchers have used the ST25R3916's active load modulation and EMD (Electromagnetic Disturbance) suppression features to build low-cost relay attack toolkits that can transparently bridge NFC transactions across distances — effectively defeating proximity assumptions in contactless payment systems. A rogue device powered by this chip can intercept a tap-to-pay transaction between a victim's card and a legitimate terminal, forwarding it to a colluding device meters away in real time.

## Key facts
- Supports ISO 14443 A/B, ISO 15693, ISO 18092 (NFCIP-1), and FeliCa — making it a universal NFC attack/defense surface
- Active Load Modulation (ALM) improves card emulation signal strength, enabling more reliable skimming and relay attacks at marginal ranges
- Integrated SPI/I²C interface means firmware-level compromise of a host MCU can directly manipulate NFC transactions without hardware modification
- Wake-on-RF feature allows dormant implanted/concealed devices to activate only when a target NFC field is detected — reducing RF detection sweep effectiveness
- The chip's multi-protocol support makes it a favorite in commercial NFC pentesting tools like Proxmark-compatible custom builds
