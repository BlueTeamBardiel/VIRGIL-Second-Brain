# Doxing

## What it is
Like a detective assembling a murder board — pulling newspaper clippings, photos, and address records until a complete picture of a person emerges — doxing is the practice of researching and publicly exposing private or identifying information about an individual without their consent. The term derives from "dropping documents" (dox) and weaponizes Open Source Intelligence (OSINT) techniques against private individuals rather than target organizations.

## Why it matters
A threat actor targeting a security researcher aggregates their LinkedIn profile, GitHub commits (revealing their real name), public voter registration records, and an old forum post containing a home address — then publishes the compiled profile to an extremist community, triggering harassment campaigns. Defenders counter this by practicing a "minimal digital footprint" strategy: auditing public data broker sites (Spokeo, Whitepages), using PO boxes, and separating professional and personal online identities.

## Key facts
- Doxing itself is not universally illegal in the U.S., but downstream actions it enables (stalking, harassment, swatting) are federal crimes under statutes like the CFAA or state anti-stalking laws
- Primary data sources used: social media OSINT, WHOIS records, data broker aggregators, breached credential databases, and public government records
- **Swatting** is a common escalation: using doxed address information to make a false emergency report, sending armed police to the victim's location
- Doxing is classified as a social engineering attack vector in the CySA+ framework because it exploits human information exposure, not technical vulnerabilities
- Removing personal data requires submitting opt-out requests to data brokers individually — no single automated removal exists, making prevention harder than remediation
