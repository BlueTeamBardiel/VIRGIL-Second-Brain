# ARO

## What it is  
Imagine a locksmith who keeps a spare key that can unlock any lock if the original key is lost. ARO—Account Recovery Operations—is that spare key for digital accounts: a controlled process that lets a user or admin regain access when credentials are lost or compromised, usually via secondary verification like an OTP or security questions.
