# Dafydd Stuttard

## What it is
Like a master locksmith who writes the definitive guide on how locks can be picked, Dafydd Stuttard (known online as "PortSwigger") is the security researcher and developer who created Burp Suite and co-authored *The Web Application Hacker's Handbook*. He is one of the most influential figures in web application security research and tooling.

## Why it matters
When a penetration tester needs to intercept, modify, and replay HTTP requests against a login portal to test for authentication bypass vulnerabilities, they almost certainly reach for Burp Suite — Stuttard's flagship tool. His work directly shaped how modern web application penetration testing is conducted, making systematic vulnerability discovery accessible and repeatable for security professionals worldwide.

## Key facts
- Founded PortSwigger Ltd., the company behind Burp Suite, which became the industry-standard web application security testing platform
- Co-authored *The Web Application Hacker's Handbook* (2nd ed., 2011) with Marcus Pinto — a canonical reference for understanding injection flaws, session management attacks, and access control vulnerabilities
- Burp Suite's Proxy component operates as an intercepting man-in-the-browser proxy, enabling real-time manipulation of web traffic between client and server
- PortSwigger's Web Security Academy (free platform) is directly relevant to understanding OWASP Top 10 vulnerabilities tested on CySA+ and related certifications
- Stuttard's research contributed significantly to understanding of advanced web vulnerabilities including HTTP request smuggling and DOM-based XSS
